# AI Development Logs

This directory contains logs of AI-assisted development sessions for the KesslerX project.
Each session is logged with the prompts given and the outputs generated, enabling
teammates to pick up where the previous developer left off without losing context.

## Structure

```
AI_LOGS/
├── README.md                 # This file
├── session-001.md            # First development session
├── session-002.md            # Second development session
└── ...
```

## How to Use

1. **Before starting work**: Read the latest session log to understand what was built and what's pending.
2. **During your session**: The AI agent will append to or create a new session log.
3. **Handoff**: Your teammate reads the latest log and their AI agent has full context.

## Session Format

Each session log contains:
- **Session metadata** (date, developer, agent model)
- **Prompts** (what was asked)
- **Decisions made** (design choices, trade-offs)
- **Files created/modified** (with rationale)
- **Current status** (what's done, what's next)
